# Interactive Star Heart

An interactive 3D heart composed of glittering stars that you can expand and contract using your mouse wheel.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```

## Build for Production

```bash
npm run build
```

## Features

- **Interactive 3D Visualization**: Beautiful star-based heart made with Three.js
- **Mouse Controls**: Click and drag to rotate, use mouse wheel to zoom
- **Reset Option**: Press Space to reset the view
- **Responsive Design**: Works on all screen sizes
